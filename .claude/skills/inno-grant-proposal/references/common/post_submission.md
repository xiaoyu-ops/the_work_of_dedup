# Post-Submission Checklist

## US Track

- [ ] Confirm submission receipt from grants.gov / Research.gov / eRA Commons
- [ ] Note review timeline (e.g., NIH: ~6-9 months from submission)
- [ ] Prepare for Just-In-Time (JIT) request if scored well
- [ ] If not discussed: plan resubmission strategy (A1 for NIH)
- [ ] Update NSF/NIH biosketch for next proposal

## CN Track

- [ ] Confirm submission on NSFC ISIS system (科学基金网络信息系统)
- [ ] Note review timeline (typically 6-8 months, results announced Aug-Sept)
- [ ] Prepare for possible phone consultation from reviewers (函评后会评)
- [ ] If not funded: analyze review comments for resubmission next cycle
- [ ] Update NSFC profile for next application year
