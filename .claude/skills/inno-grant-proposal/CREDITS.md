## How This Skill Differs

While this skill draws inspiration from the above sources, it is an original work that:
- Integrates US and CN grant systems into a single dual-track workflow
- Introduces the "Adapt from Previous Proposal" workflow for the most common use case
- Combines deterministic script-based checks with AI semantic analysis (Tier 1 + Tier 2)
- Provides bilingual support with language-appropriate AI-flavor detection
- Implements lazy loading for optimized token usage
- Includes original validation scripts, rubric JSON schemas, and template designs
